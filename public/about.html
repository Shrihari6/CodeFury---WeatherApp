About Me & This Project

Hello! I’m Shrihari Jawalgi, currently pursuing my degree in Information Science and Engineering. This Weather App — CloudCtrl — was built as part of the IEEE UVCE CodeFury 8.0 hackathon.

The goal was to create a fast, visually appealing, and user-friendly tool that delivers real-time weather updates for any location worldwide. The project uses Node.js, HTML, CSS, and JavaScript with the OpenWeatherMap API, along with modern UI styling and animations to make it a delightful experience.

I have completed several relevant certifications in web development and related technologies, and I have built other projects you can explore on my GitHub. I have also completed two internships where I applied my skills in real-world scenarios, gaining hands-on experience in both frontend and backend development.

Through this project, I aimed to combine functionality with creativity — from fetching live data to implementing smooth animations — while keeping the app responsive and easy to use for everyone.
